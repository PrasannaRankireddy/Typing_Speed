A simple Typing Speed Test application built using Python and Tkinter. This project helps users improve their typing speed and accuracy by measuring how fast and accurately they can type a given sentence.

**Features**

User-friendly GUI with Tkinter

Displays a sample sentence for typing

Measures typing speed (words per minute - WPM)

Calculates accuracy of typed text

Shows time taken to complete typing

Reset and try again functionality
**How to Use**

Open the application.

Read the sample sentence displayed.

Start typing in the input box.

Click Submit or press Enter when finished.

The app will display:

Typing Speed (WPM)

Accuracy (%)


Time Taken (seconds)

Click Reset to try again.
